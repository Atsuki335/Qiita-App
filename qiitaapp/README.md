# qiitaapp

FlutterによるQiitaクライアント

https://www.flutter-study.dev/create-app/qiita-app

## 環境

* fvm
* VSCode
* Xcode14.2
* Android Studio 2022.2.1

iOSシミュレーターで動作確認

## 構築手順

1. fvmをインストール

```
$ brew tap leoafarias/fvm
$ brew install fvm
```

2. fvmで指定されたバージョンでflutterをインストール

```
$ fvm install 2.0.1
```

3. VSCodeもしくはコマンドラインで起動

iOSシミュレーターを起動

## VSCode

実行とデバッグ - Qiita App

## コマンドライン

```
$ fvm flutter run -t lib/main.dart
```
